## 健身笔记
```
维生素B族+坚果（杏仁）
热身动作：双手拉住杠杆，向上提身体

练习腿部力量（每周一次）：
深蹲（10,15kg共12组）
提哑铃走步
器械抬腿
调节腿部放松方式：双曲腿后仰，单曲腿拉伸
++双手拉直杆（半蹲，手臂保持伸直）--练背厚度
++双手快速举哑铃（10LB，4组，中间休息30秒）
----------------------------------

练背部（不要都练，每周换几样练就行）：
杠铃曲腿硬拉（右手正，左手反握）
背阔肌下拉
一手扶凳单臂哑铃划船
背中束下拉
++双手拉直杆（半蹲，手臂保持伸直）--练背厚度
半蹲拉杠铃（沿大腿方向提拉）
双手向两侧拉（手臂保持伸直）
坐姿拉力器划船
----------------------------------

练胸和前束肩部：
平卧推
上斜卧推
下斜卧推（哑铃）
哑铃微斜举
夹胸（最后做）

练中后束肩部：
哑铃平举（90度）

++双手拉直杆（半蹲，手臂保持伸直）--练背厚度
++双手快速举哑铃（10LB，4组，中间休息30秒）
-----------------------------------

练二头肌：
加紧胳膊的那个器械（6组）
坐在扶凳上双手交替举哑铃（6组）
蹲拉直杆（6组）
++双手拉直杆（半蹲，手臂保持伸直）--练背厚度
++双手快速举哑铃（10LB，4组，中间休息30秒）
```
